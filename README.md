# pd-todo

A simple to-do list for the Playdate console. Eventually might flesh out into something resembling a PDA.

## Features

### To Dos

- [x] Basic to-do list functionality
  - [ ] Add/remove to-dos
  - [x] Mark to-dos as complete/incomplete
- [ ] Detail view for individual to-dos
- [ ] Complex to-do management
  - [ ] Categories/tags
  - [ ] Priorities
- [x] Save/load to-dos using Playdate's datastore
- [x] Simple UI for displaying and interacting with to-dos
- [x] Fetch to-dos from a remote server (mock API for now)
- [ ] Improve UI/UX
  - [ ] Replace checkboxes with sprites
  - [ ] Add animations for syncing
- [ ] Add due dates

### Calendar

- [ ] Integrate a simple calendar view to see to-dos by date
- [ ] Sync with external calendar services (maybe)

### Phone Book

- [ ] View contacts

### Voice Memos

- [ ] Record and playback voice memos
- [ ] Organize memos by date or tags
- [ ] Sync memos with cloud storage
